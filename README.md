# Assignment_week_2

Java 17 Maven project in IntelliJ IDEA.

Implements 2 classes ReverseAndSearchArray.java and StringBuilderManipulation.java

## ReverseAndSearchArray.java

Basic operations with arrays, using Scanner and StringBuilder.

Execution example: 
```bash
  How many numbers do you want to enter?: 
  6
  Enter number 1: 
  4
  Enter number 2: 
  3
  Enter number 3: 
  6
  Enter number 4: 
  1
  Enter number 5: 
  6
  Enter number 6: 
  6
  The array is: [ 4 3 6 1 6 6 ]
  The reversed array is: [ 6 6 1 6 3 4 ]
  Give me a number to search in the array: 
  6
  The number 6 is found in the array in the index: 2 4 5 
```
